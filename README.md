# 模式识别大作业 — 大学生生成式 AI 使用频率分类模型

## 📋 项目简介

本项目基于 **1781 份大学生生成式人工智能（AIGC）使用情况问卷**数据，构建完整的机器学习分类 Pipeline，将大学生自动分为「**低频 / 中频 / 高频**」三类 AI 使用者，并深入分析**影响使用频率的关键因素**。

### 核心研究问题

> 哪些因素决定了大学生使用生成式 AI 频率的高低？

---

## 🔬 实验设计

### 实验 1：多模型对比

使用 **SMOTE 过采样** 解决类别不平衡问题，训练并对比以下 6 个分类模型：

| 模型 | Macro F1 | CV Mean |
|------|----------|---------|
| **SVM** | **0.4988** | **0.4860** |
| Logistic 回归 | 0.4951 | 0.5134 |
| 随机森林 | 0.4761 | 0.4898 |
| 朴素贝叶斯 | 0.4125 | 0.4024 |
| 决策树 | 0.3900 | 0.4368 |
| KNN | 0.3622 | 0.3765 |

- 采用 **80/20 分层划分**（Stratified Split）
- **5 折交叉验证**进行模型选择
- **Macro F1** 作为主要评估指标
- SMOTE 仅在训练集 Pipeline 内部执行，测试集保持原始分布

### 实验 2：超参数调优

对 4 个关键模型使用 **GridSearchCV** 进行网格搜索调优：

- **随机森林**：n_estimators, max_depth, min_samples_split, max_features
- **SVM**：C, kernel, gamma
- **KNN**：n_neighbors, weights, metric
- **决策树**：max_depth, min_samples_split, criterion

### 实验 3：因子重要性分析

综合使用 **三种方法** 交叉验证影响使用频率的关键因素：

1. **随机森林特征重要性**（Gini Importance）
2. **Permutation Importance**（排列重要性）
3. **SHAP 值分析**（SHAP Bar / Summary / Beeswarm）

通过**共识排名**（三方法平均排名）确定最终 Top-10 关键因素：

| 排名 | 特征 | 平均排名 |
|------|------|----------|
| 1 | 使用产品数量 (product_count) | 1.3 |
| 2 | 文本生成功能使用 | 3.0 |
| 3 | 文心一言使用 | 4.0 |
| 4 | 性别 | 7.7 |
| 5 | ChatGPT 使用 | 8.3 |

### PCA 降维可视化

对特征进行 PCA 降维至 2 维，观察三类用户在降维空间中的分布：
- PC1 解释方差：17.4%
- PC2 解释方差：5.8%
- 总计：23.3%

## 📝 数据处理说明

### 排除的列

| 排除列 | 排除原因 |
|--------|----------|
| 序号 | 问卷编号，无意义 |
| 学校 | 与专业高度共线，且结论不应是"某校更爱用 AI" |
| 所用时间 | 问卷元数据，与使用频率无因果关系 |
| 总分 | 量表总分，与目标变量高度相关（目标泄漏） |
| Q15（总体情况） | 目标泄漏："离不开 AI"与 Q9 相关系数 r=0.235 |
| Q16（该不该使用） | 近零方差：95.9% 选"应该" |
| Q17、Q18 | 开放文本题 |
| Q14 注意力检测题 | 34.9% 未通过，从特征中排除 |

### 专业映射方案

本科和专科使用不同的专业编码体系，统一映射为：
- **理科**：理学、工学、农学、医学等
- **文科**：哲学、经济学、法学、教育学、文学、历史学、管理学、艺术学等
- **其他**

## 📌 关键发现

1. **使用产品数量**是最强预测因子（r=0.364），使用越多不同 AI 产品，频率越高
2. **文本生成功能**的使用是区分使用频率的重要因素
3. **性别**对使用频率有显著影响
4. SVM 在该数据集上表现最优（Macro F1 = 0.4988），但整体分类难度较大，主要受限于低频组样本量不足（8.1%）
5. PCA 显示三类用户在降维空间中存在一定重叠，说明使用频率受多因素共同影响
