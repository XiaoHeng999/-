# 大学生 AIGC 使用频率分类 — 项目上下文

## 数据源
- `数据.xlsx`：1781 条问卷数据，78 列
- `ai使用文档.docx`：问卷题目与选项说明

## 目标
预测大学生 AIGC 使用频率分类（低/中/高），并进行特征重要性分析。

### 目标变量 (Q9 → 三分类)
| 编码值 | 原始含义 | 分组 |
|--------|----------|------|
| 1 | 从未使用 | 低频 (145人, 8.1%) |
| 2 | 一周1~10次 | 中频 (1315人, 73.8%) |
| 3~6 | 一周11次以上 | 高频 (321人, 18.0%) |

### 特征池 (64个特征)
| 特征组 | 列 | 编码方式 |
|--------|------|----------|
| 性别 (Q1) | 二值 (1=男, 2=女) | 保持原编码 |
| 学历 (Q2) | 1=本科, 2=专科, 3=博士 | One-Hot |
| 专业 (Q3) | → 文/理/其他 | One-Hot (本科/专科编码体系统一映射) |
| 年级 (Q4) | 1~5 | One-Hot |
| 课程偏好 (Q5) | 4选项 | One-Hot |
| 课堂知识重要度 (Q6) | 0~100% 数值 | 数值型 |
| 分数重要度 (Q7) | 0~100% 数值 | 数值型 |
| 产品使用 (Q8) | 10个二值列 + 产品数量 | 原编码 |
| 功能排序 (Q10) | 13列排序, NaN→14 | 数值型 |
| 专业学习 (Q11) | 9题 Likert 1~5 | 数值型 |
| 科研活动 (Q12) | 9题 Likert 1~5 | 数值型 |
| 日常生活 (Q13) | 8题 Likert 1~5 | 数值型 |
| 升学就业 (Q14) | 7题 Likert 1~5 (排除注意力检测题) | 数值型 |

### 排除的列
- 序号、学校、所用时间、总分 (元数据/泄漏)
- Q8 其他产品 (可选，但含在10列内)
- Q15 总体情况 (目标泄漏 r=0.24~0.36)
- Q16 该不该使用 (方差接近零 95.9%选"应该")
- Q17, Q18 (开放文本题)
- 注意力检测题 (14题中的"请选比较同意")

### 建模决策
- **采样策略**: SMOTE 过采样 (仅训练集，Pipeline 内)
- **数据划分**: 80/20 stratified split + 5折交叉验证
- **评估指标**: Macro F1 为主 + 每类 Precision/Recall + 混淆矩阵
- **模型**: KNN、决策树、随机森林、SVM、朴素贝叶斯、Logistic回归

### 分析与可视化 (PPT级别)
- PCA 降维可视化
- 6模型 Macro F1 对比
- 随机森林特征重要性
- SHAP 值分析
- Permutation Importance
- 所有图表 PPT 可直接使用
